博客开通！

2020年2月27日
| 生活
| 阅读

很早之前就想搭建自己的个人blog，因为工作业务原因，没有把时间安排在个人成长这些事情上面。

但是，经历了入行的这三年，越发的觉得技术实力的成长是离不开思考、实践和总结的。

所以，现在新建立起我的个人blog。 blog使用的比较简单的实现 Hugo + GitHub Pages.

未来打算自己学习过程中总结性的内容和一些使用教程放进blog中。

希望自己的水平能随着博客一起成长。

2020加油！！！！
